Features

Creates a bar on top of each editor for manage tabs, with a button for toggle sidebar, and another to see a list of tabs.
  • drag & drop tabs
  • drag & drop filePath
  • button flip view
  • button close file (include middle click)
  • information about file in tooltip
  • It contains icons of your choice
  • enable opacity of the panel
  • icons in gray scale mode
  • a tab menu with multiple buttons
  • {{{MMB}}}
  • add/retrive bookmarks by click on gutter
  • [ContextMenu] copy Path File
  • {{{CWMT}}}

Also has an option to display the sidebar when the mouse is over the element. More Info,Report a Bug
